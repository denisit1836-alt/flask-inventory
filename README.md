# Учет товаров на Flask

## Описание проекта

Учет товаров — это простое веб-приложение на Flask для ведения учета товаров, продаж и услуг в небольшом магазине или складе.

## Основные функции

- **Товары** — добавление, редактирование, удаление товаров (название, категория, производитель, страна, цена, количество).
- **Продажи** — регистрация продаж с выбором товара и услуги, автоматическим расчетом общей стоимости (цена товара + цена услуги).
- **Услуги** — справочник услуг (название, описание, цена).
- **Авторизация** — вход в систему, регистрация.

Удобный интерфейс с выпадающими списками, автоподстановкой цены, календарем для даты.

Приложение создано в образовательных целях и демонстрирует базовые принципы работы с Flask, SQLAlchemy, Flask-Login и Jinja2.

## Технологии и библиотеки

- Python 3.10+
- Flask — веб-фреймворк
- Flask-SQLAlchemy — работа с базой данных
- Flask-Login — управление авторизацией
- MySQL — база данных (через драйвер pymysql)
- Jinja2 — шаблонизатор
- HTML5 / CSS — фронтенд (чистый, без фреймворков)
- JavaScript — минимальный клиентский скрипт для автоподстановки цены

## Особенности

- Автоматический расчёт стоимости в продажах (товар + услуга)
- Выпадающие списки с данными из связанных таблиц
- Календарь для выбора даты
- Защита всех маршрутов авторизацией
- Flash-сообщения для обратной связи

## Дальнейшее развитие

## Установка и запуск

### Требования
- Python 3.10 или выше
- MySQL сервер (локальный или удалённый)

### Клонируйте репозиторий
   ```bash
   git clone https://github.com/denisit1836-alt/flask-inventory.git
   cd flask-inventory
   ```
### Создайте виртуальное окружение
   ```bash
   python -m venv venv

   venv\Scripts\activate
```
### Установите зависимости

   ```bash
   pip install -r requirements.txt
```
### Настройте подключение к базе данных

- Откройте файл app.py и измените строку с подключением:

   ```bash
   app.config['SQLALCHEMY_DATABASE_URI'] = 'mysql+pymysql://ваш_логин:ваш_пароль@localhost/имя_вашей_базы'
   ```
### Запустите приложение

- python app.py

- Откройте в браузере

- Перейдите по адресу:

- http://127.0.0.1:5000
